# Simworld

A 2D simulation game built with Phaser.js featuring an animated character in a multi-story building environment.

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the game:
   ```bash
   npm start
   ```

3. Open `http://localhost:8080` in your browser

## Controls

- **A/D**: Move left/right
- **ESC**: Quit
